# Emberwake Idle

A lightweight mobile-first idle RPG prototype inspired by games like Melvor Idle.

## What is included

- Auto-battling combat with enemy progression
- Idle gathering skills
- Crafting for consumables and permanent upgrades
- Persistent local save data with offline gains
- A single-screen mobile-friendly interface

## Run it

Open `index.html` in a browser. No build step or package install is required.

## Publish to GitHub Pages

This repo includes a GitHub Pages workflow at `.github/workflows/deploy-pages.yml`.

To publish it:

1. Push this project to a GitHub repository.
2. Open `Settings > Pages` in that repo.
3. Set the Pages source to `GitHub Actions`.
4. Push to `main` or `master`, or run the workflow manually from the `Actions` tab.

After the workflow finishes, GitHub will show the public Pages URL you can share with testers.
